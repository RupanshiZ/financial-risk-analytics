# Data

## Raw data (not included in this repository)

The raw Lending Club CSV files are **not committed to Git** due to their size
and Kaggle's terms of use. You must download them yourself.

### Download instructions

1. Go to Kaggle:
   https://www.kaggle.com/datasets/wordsforthewise/lending-club

2. Download both files and place them in `data/raw/`:

```
data/raw/
├── accepted_2007_to_2018q4.csv      (~1.8 GB, ~2.26 million rows)
└── rejected_2007_to_2018q4.csv      (~0.5 GB, ~27 million rows)
```

### Field documentation

The accepted file contains loan-level data for all originated loans
2007–2018 Q4. The rejected file contains summary data on applications
that were declined.

Full data dictionary:
https://www.kaggle.com/datasets/wordsforthewise/lending-club

## Processed data

`data/processed/lending_club.duckdb` is generated by running:

```bash
python scripts/load_lending_club_data.py
```

This file is also excluded from Git (it can always be rebuilt from the
raw CSVs). It contains two clean analytical tables:
- `accepted_loans`       — all originated loans with cleaned types
- `rejected_applications` — declined applications
